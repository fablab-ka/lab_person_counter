# Lab Person Counter

A simple ESP8266 and MAX7219 based contraption to easily let people count themselves when entering the Makerspace.
Tested with an nodemcu chip

# Wiring

Increase Button

    D4 and GND

Decrease Button

    D3 and GND

MAX7219 panels (at least 4)

    CLK_PIN:		     D5
    DATA_PIN:	       D7
    CS_PIN:		       D8
    VCC              5V
    GND              GND
